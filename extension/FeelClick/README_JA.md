# FeelClick - 片手でのウェブスクロール体験

[English](README.md) | [Tiếng Việt](README_VI.md)

> **バージョン**: 0.1.0 Beta

FeelClickは、片手だけで簡単にウェブを閲覧したり、マウスホイールが壊れたりしたユーザーを支援するために設計されたChrome拡張機能です。スマートな「スクロールゾーン」を使用すると、シンプルなマウスクリックでスムーズなスクロール操作を実行できます。

## 🌟 主な機能

- **スクロールゾーン**: 画面上のどこにでも移動して配置できる透明なオーバーレイ。
- **5つの柔軟なスクロールモード**: 習慣に合わせて左/右クリックのアクションをカスタマイズできます。
- **ドラッグ＆ドロップと位置ロック**: スクロールゾーンを便利な位置に簡単に移動し、誤操作を防ぐためにロックできます。
- **モダンなインターフェース**: 視界を遮らない、洗練されたグラスモーフィズムデザイン。
- **設定の保存**: 位置と設定を自動的に記憶します。

## 🎮 コントロールモード

この拡張機能は5つの異なるコントロールモードを提供し、ポップアップからすばやく変更できます。

1.  **モード 1 (デフォルト)**:
    - 左クリック: 上にスクロール
    - 右クリック: 下にスクロール
2.  **モード 2**:
    - 左クリック: 下にスクロール
    - 右クリック: 上にスクロール
3.  **モード 3 (左右分割)**:
    - ゾーンの左半分をクリック: 上にスクロール
    - ゾーンの右半分をクリック: 下にスクロール
4.  **モード 4 (上下分割)**:
    - ゾーンの上半分をクリック: 上にスクロール
    - ゾーンの下半分をクリック: 下にスクロール
5.  **モード 5 (連続スクロール)**:
    - 左クリックを長押し: 徐々に下にスライド
    - 右クリックを長押し: 徐々に上にスライド

## 🚀 インストールガイド

1.  FeelClickのソースコードをダウンロードします（またはこのリポジトリをクローンします）。
2.  Chromeブラウザを開き、`chrome://extensions/` にアクセスします。
3.  右上の **デベロッパーモード** をオンにします。
4.  **パッケージ化されていない拡張機能を読み込む** ボタンをクリックします。
5.  FeelClickのソースコードが含まれているフォルダ（`manifest.json` ファイルが含まれているフォルダ）を選択します。
6.  これで拡張機能を使用する準備が整いました！

## 💡 使用ガイド

1.  インストール後、ツールバーにFeelClickアイコンが表示されます。
2.  任意のウェブサイトを開きます（または現在のページをリロードします）。
3.  半透明のボックス（スクロールゾーン）が表示されます（デフォルトは右隅）。
4.  **移動**: スクロールゾーンで左マウスボタンをクリックしたままドラッグして位置を変更します。
5.  **ロック/ロック解除**: スクロールゾーンの小さな鍵アイコンをクリックして位置を固定します。
6.  **モード変更**: ツールバーの拡張機能アイコンをクリックして設定メニューを開き、目的のモードを選択します。

## 📂 フォルダ構成

- `manifest.json`: 拡張機能のメイン設定ファイル。
- `popup.html`, `popup.css`, `popup.js`: 設定ウィンドウのインターフェースとロジック。
- `content.js`, `content.css`: ウェブサイトに埋め込まれたスクロールゾーンのスクリプトとインターフェース。
- `icons/`: アプリケーションアイコンを含むフォルダ。
