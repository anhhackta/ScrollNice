Changelog

v3.0-design — 2026-05-08

Slint UI Migration Design approved — thống nhất roadmap chuyển từ C++/Win32 sang Rust/Slint với Fluent UI.

  • ✅ Thiết kế kiến trúc 4 lớp: Slint UI, Rust bridge, core logic, platform layer
  • ✅ Chốt hệ Theme light/dark và i18n runtime EN/VI
  • ✅ Chốt preset/profile schema và luồng lưu cấu hình
  • ✅ Chốt wheel-block modes + auto-detect scroll target
  • ✅ Cập nhật website/docs để phản ánh lộ trình migration

v2.1 — 2025 (Hiện tại / Current)

3-Mode Redesign — Đơn giản hóa từ 5 xuống 3 chế độ, UX/UI được cải thiện đáng kể.

  • ✅ Mode 1: Click/Giữ — Click trái/phải + giữ để lăn liên tục với gia tốc tự nhiên
  • ✅ Mode 2: Chia Trên/Dưới — Click hoặc giữ mỗi nửa zone
  • ✅ Mode 3: Di Chuột Tự Động — Hover vào zone = tự lăn, không cần click! (Mode mới hoàn toàn)
  • ✅ Visual zone cải thiện cho từng mode với nhãn tiếng Việt + English
  • ✅ WM_MOUSELEAVE tracking cho Mode 3 (hover auto dừng khi chuột rời zone)
  • ✅ Extension: 3 mode cards; bilingual popup + options page
  • ✅ Website/Docs: Toàn bộ hướng dẫn song ngữ Việt-Anh
  • ✅ Config.json: thêm trường hover_speed

v2.0 — 2025

Kiến trúc mới — Chuyển từ edge-zone sang floating zone.

  • Zone nổi always-on-top thay thế edge scrolling cũ
  • 5 chế độ lăn (đã gộp xuống 3 ở v2.1)
  • Cài đặt đầy đủ trong tray icon menu
  • Âm thanh click via Web Audio API
  • Chặn wheel vật lý với phím thoát Alt
  • Resize handle cho zone
  • Extension: glassmorphism UI, popup và options page mới
  • CMake build system

v1.0 — FeelClick

Phiên bản Chrome Extension đầu tiên (FeelClick) — zone cố định, click trái/phải cơ bản.
