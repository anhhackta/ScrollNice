Cài đặt / Installation

🖥️ Ứng dụng Windows (Portable)

  1. Tải ScrollNice-v2.zip từ trang Releases
  2. Giải nén vào bất kỳ thư mục nào
  3. Chạy ScrollNice.exe
  4. Zone lăn sẽ xuất hiện — sẵn sàng sử dụng!

Cấu hình lưu tại config.json cùng thư mục với file .exe.

Khởi động cùng Windows / Start with Windows

Click phải tray icon → Settings → tích "Khởi động cùng hệ thống".

🌐 Chrome Extension

  1. Tải hoặc clone folder extension
  2. Mở Chrome → vào chrome://extensions/
  3. Bật Developer mode (nút toggle góc trên phải)
  4. Click Load unpacked
  5. Chọn thư mục extension/FeelClick
  6. Icon ScrollNice xuất hiện trên toolbar

Yêu cầu hệ thống / Requirements

Thành phần Yêu cầu
Hệ điều hành Windows 10 / 11 (64-bit)
RAM < 8 MB
CPU ≈ 0% khi không dùng
Dung lượng < 1 MB
Chrome (Extension) v88+
