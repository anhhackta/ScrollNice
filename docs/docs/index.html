Hướng dẫn ScrollNice

Chào mừng! ScrollNice là tiện ích miễn phí giúp bạn lăn trang bằng cách click hoặc di chuột vào một zone nổi — không cần dùng wheel chuột.

Có trên nền tảng

Nền tảng Mô tả (VI) Description (EN)
🖥️ Windows App File .exe portable, hoạt động toàn hệ thống Portable .exe, works system-wide
🌐 Chrome Extension Chỉ trong trình duyệt, không cần cài đặt chính thức Browser-only, load as unpacked

3 Chế độ lăn

Mode Mô tả (VI) Description (EN)
Mode 1 Click trái/phải + giữ → lăn liên tục có gia tốc L/R click + hold for continuous scroll
Mode 2 Zone chia trên/dưới, click/giữ mỗi nửa Top/bottom split, click or hold each half
Mode 3 Di chuột vào zone → tự lăn, không cần click Hover to auto-scroll, no click needed
