Config File (Windows)

All settings are stored in config.json next to the exe. You can edit this file directly with any text editor. Changes take effect on restart.

{
  "version": 1,
  "enabled": true,
  "start_with_windows": false,
  "wheel_block": false,
  "zone": { "x": 100, "y": 100, "width": 120, "height": 180, "opacity": 0.25 },
  "scroll": { "mode": "click_lr", "scroll_amount": 300 },
  "sound": { "enabled": true },
  "hotkeys": { "toggle_enabled": "Ctrl+Alt+S" }
}
