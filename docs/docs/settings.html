Cài đặt chi tiết / Settings Reference

Chung / General

Cài đặt Mặc định Mô tả
Bật/Tắt Bật Bật/Tắt toàn bộ ứng dụng
Khởi động cùng Windows Tắt Tự chạy khi đăng nhập Windows (registry key)
Chặn Wheel Tắt Chặn sự kiện wheel vật lý. Nhấn Alt để bỏ qua chặn nhất thời

Chế độ lăn / Scroll Mode

Cài đặt Mặc định Mô tả
Chế độ Mode 1 Xem Chi tiết 3 chế độ
Scroll Amount 300 px Số pixel lăn mỗi lần click (50–800)
Tốc độ giữ 8 px/tick Tốc độ cơ bản khi giữ chuột
Gia tốc giữ 3 Tăng tốc mỗi giây khi giữ
Tốc độ hover 6 px/tick Tốc độ cuộn ở Mode 3 di chuột

Zone

Cài đặt Mặc định Mô tả
Rộng × Cao 120 × 200 px Kích thước zone (tối thiểu 60px mỗi chiều)
Độ trong suốt 30% Mức trong suốt (5–100%)
Màu sắc #3498db Màu nền zone (chỉ ứng dụng Windows)
Ảnh bìa Không có Đường dẫn ảnh tùy chỉnh cho zone
Khoá vị trí Tắt Khóa zone, không thể kéo nhầm

Âm thanh / Sound

Cài đặt Mặc định Mô tả
Âm thanh click Bật Phát âm thanh mỗi lần click trong zone
File âm thanh Tích hợp Đường dẫn file WAV tùy chỉnh (để trống = dùng mặc định)

Phím tắt / Hotkeys (Windows)

Chức năng Mặc định
Bật/Tắt ScrollNice Ctrl+Alt+S
Chế độ chỉnh sửa Ctrl+Alt+E
Bật/Tắt chặn Wheel Ctrl+Alt+W

File config / Config File

Tất cả cài đặt lưu ở config.json cùng thư mục exe. Có thể chỉnh trực tiếp bằng text editor:

{
  "scroll": {
    "mode": "click_hold",
    "scroll_amount": 300,
    "continuous_speed": 8,
    "continuous_accel": 3,
    "hover_speed": 6
  },
  "zone": { "x": 60, "y": 100, "width": 120, "height": 200, "opacity": 0.30 }
}
