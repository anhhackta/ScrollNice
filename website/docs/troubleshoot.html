Troubleshooting

Zone not visible

  • Ensure ScrollNice is running (check system tray)
  • Press Ctrl+Alt+S to toggle on
  • Check if opacity is too low in Settings

Scrolling doesn't work

  • Make sure you're clicking inside the zone
  • Check that the correct mode is selected
  • Some apps with custom scroll handling may not respond to SendInput wheel events

Zone won't move

  • Check if "Lock Position" is enabled ‚Üí unlock it in Settings
  • Enter Edit Mode with Ctrl+Alt+E

Extension not working on a page

  • Extensions cannot run on chrome:// pages or the Chrome Web Store
  • Try reloading the page after installing
  • Check that the extension is enabled in chrome://extensions

Wheel block is stuck

  • Panic key: Hold Alt while scrolling to bypass the block
  • Press Ctrl+Alt+W to toggle wheel block off
  • Press Ctrl+Alt+S to disable ScrollNice entirely
